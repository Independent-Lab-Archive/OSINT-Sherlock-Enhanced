# OSINT-Sherlock-Enhanced
Enhanced Sherlock script for automated OSINT username reconnaissance across multiple platforms.

## Overview
Customized Sherlock script to expand username reconnaissance across multiple platforms. Designed for automated, ethical OSINT data collection in lab and research environments.

## Tools Used
- Python 3  
- Sherlock (original script)  
- APIs & web automation  

## What I Did
- Added support for additional websites and platforms  
- Fixed script errors and improved reliability  
- Automated username scanning for multiple targets  
- Compiled and analyzed results for actionable insights  

## Key Results
- Increased coverage and accuracy of OSINT recon  
- Reduced manual effort in reconnaissance tasks  
- Enhanced understanding of online footprint mapping  

## Notes
- Testing performed ethically using non-sensitive, public usernames  
- Script is intended for research and educational purposes
